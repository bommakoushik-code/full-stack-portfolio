# Full Stack Portfolio

A modern full-stack portfolio website built using React, Node.js, Express, and MongoDB.

## Live Demo

Frontend: https://full-stack-portfolio-mu-ten.vercel.app

## Features

* Responsive portfolio design
* Modern React UI
* Contact form integration
* MongoDB database storage
* Express.js REST API
* Resume download
* GitHub and LinkedIn links

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Lucide React

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Installation

### Frontend

```bash
cd portfolio-react
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```

## Contact Form Workflow

User → React Frontend → Express Backend → MongoDB Atlas

Messages submitted through the contact form are securely stored in MongoDB Atlas.
